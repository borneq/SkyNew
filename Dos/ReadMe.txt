use coprocessor, speedup and compare results to Delphi/Lazarus version.
To compile this:
- install DOSBox
- in c:\Users\USER\AppData\Local\DOSBox\dosbox-0.74.conf change last lines to:
[autoexec]

# You can put your MOUNT lines here.
MOUNT C D:\DOSBOX
